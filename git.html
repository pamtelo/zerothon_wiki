Git

Git 설치 및 기본 설정

  1. Git 설치
    • Windows: Git for Windows 다운로드 및 설치
    • Mac: Homebrew를 통한 설치 또는 공식 설치 파일 사용
  2. 기본 설정
    • 사용자 이름 설정: git config --global user.name "Your Name"
    • 이메일 설정: git config --global user.email "your.email@example.com"
  3. 기본 명령어
    • git init: 새 저장소 생성
    • git clone: 원격 저장소 복제
    • git add: 파일 스테이징
    • git commit: 변경사항 커밋
    • git push: 원격 저장소에 푸시
    • git pull: 원격 저장소에서 변경사항 가져오기
